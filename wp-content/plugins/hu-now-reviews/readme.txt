== Description ==
A lightweight, native reviews plugin for HU NOW. Provides shortcodes for ratings, reviews, and submission form with built-in styling and JSON-LD schema.

== Shortcodes ==
[hu_rating_summary] - Show average rating and stars
[hu_reviews] - Display user reviews list
[hu_review_form] - Display the submission form

== Installation ==
1. Upload `hu-now-reviews.zip` via Plugins → Add New → Upload.
2. Activate.
3. Insert shortcodes with Elementor Shortcode widget or directly in content.
